Smart Health Advisor is a machine learning-based application designed to assist users in managing their health by recommending suitable medicines, diets, precautionary measures, and workout routines based on symptoms or diagnosis. The project aims to offer a holistic health recommendation system that can be used by healthcare providers, patients, or wellness platforms.

The core of the system is built using Python and trained on structured datasets containing information about medications, dietary plans, and fitness guidelines. Multiple machine learning models are implemented and compared to determine the most accurate approach for classifying patient data and generating recommendations.

The classification models used include:

Support Vector Machine (SVC)

Random Forest Classifier

Gradient Boosting Classifier

K-Nearest Neighbors (KNN)

Multinomial Naive Bayes

Each model is trained and tested using accuracy scores and confusion matrices to evaluate performance. The project uses scikit-learn for machine learning algorithms, and NumPy for numerical operations.

Additionally, the system integrates static web pages created with HTML to present the user interface and relevant content like the about page, contact form, developer info, and blog section.

The input to the system is typically a set of symptoms or encoded diagnostic data, and the output includes a predicted medicine recommendation along with tailored advice on diet and physical activity. This makes the application suitable for clinical decision support, early health screening tools, or educational platforms
